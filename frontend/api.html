Документация по использованию API

Загрузка файла

POST /api/upload
Content-Type: multipart/form-data

form-data:
  file: (ваш файл)

Ответ:
{
    "success": true,
    "fileUrl": "http://domain.com/uploads/filename.ext",
    "shortUrl": "http://domain.com/f/shortId"
}

Пример на cURL:

curl -F "file=@file.jpg" http://localhost:3000/api/upload

Пример на Python:

import requests

files = {'file': open('file.jpg', 'rb')}
response = requests.post('http://localhost:3000/api/upload', files=files)
print(response.json())

Пример на JavaScript:

const formData = new FormData();
formData.append('file', fileInput.files[0]);

fetch('http://localhost:3000/api/upload', {
    method: 'POST',
    body: formData
})
.then(response => response.json())
.then(data => console.log(data));

Ограничения

  • Максимальный размер файла: 100MB
  • Запрещенные типы файлов: .exe, .php, .js и другие исполняемые файлы
